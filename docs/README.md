# 介绍

## imgAutoSize 是什么

<p class="tip">
    imgAutoSize是一个纯原生js的超轻量级图片自适应插件，不依赖任何库运行
</p>

## 作用：
<p class="warn">
 将图片以父级框架根据图片焦点位置自适应放大
</p>
